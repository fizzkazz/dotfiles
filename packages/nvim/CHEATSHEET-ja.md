# Neovim チートシート（印刷用）

対象: AstroNvim + TypeScript (`vtsls`)  
前提: `Leader = <Space>`

---

## 1. 記法

| 記法 | 意味 |
|---|---|
| `<Space>` | スペースキー |
| `<C-x>` | `Ctrl + x` |
| `:cmd` | Normalモードで `:` を押して実行 |

---

## 2. モード

| キー | 動作 |
|---|---|
| `i` | Insertモード |
| `Esc` | Normalモードへ戻る |
| `v` | 文字単位の選択 |
| `V` | 行単位の選択 |

---

## 3. 保存・終了

| キー | 動作 |
|---|---|
| `<Space>w` | 保存 |
| `<Space>q` | 現在ウィンドウを閉じる |
| `<Space>Q` | Neovimを終了 |
| `:wq` | 保存して終了 |

---

## 4. 移動

| キー | 動作 |
|---|---|
| `w` / `b` | 次/前の単語へ移動 |
| `0` / `$` | 行頭/行末 |
| `gg` / `G` | ファイル先頭/末尾 |
| `<C-o>` / `<C-i>` | ジャンプ履歴の戻る/進む |

---

## 5. 編集

| キー | 動作 |
|---|---|
| `u` | Undo |
| `<C-r>` | Redo |
| `dd` | 行削除 |
| `yy` | 行コピー |
| `p` | 貼り付け |

---

## 6. ウィンドウ操作

| キー | 動作 |
|---|---|
| `|` | 縦分割 |
| `\\` | 横分割 |
| `<C-h>` | 左ウィンドウへ |
| `<C-j>` | 下ウィンドウへ |
| `<C-k>` | 上ウィンドウへ |
| `<C-l>` | 右ウィンドウへ |
| `<C-w>j` | 下ウィンドウへ（Vim標準移動） |

---

## 7. 参照一覧（Quickfix）

`<Space>lR` で参照一覧を開いた後に使う:

| キー | 動作 |
|---|---|
| `<Space>xq` | Quickfix一覧へフォーカス |
| `]q` | 次の参照へ |
| `[q` | 前の参照へ |
| `Enter` | 参照先へジャンプ |
| `:cclose` | Quickfixを閉じる |

---

## 8. TypeScript LSP (`vtsls`)

| キー | 動作 |
|---|---|
| `gd` | 定義へジャンプ |
| `gD` | 宣言へジャンプ |
| `gy` | 型定義へジャンプ |
| `<Space>lR` | 参照一覧 |
| `<Space>lr` | リネーム |
| `<Space>la` | コードアクション |
| `<Space>lA` | Source Action（import整理など） |
| `<Space>lf` | フォーマット |
| `:LspInfo` | LSP状態確認 |

---

## 9. 困ったとき

| キー/コマンド | 動作 |
|---|---|
| `<Space>` | 使えるキーをガイド表示（which-key） |
| `:checkhealth vim.lsp` | LSPの状態確認 |

---

## 10. 最短ワークフロー（TS）

1. `gd` で定義へ  
2. `<Space>lR` で参照を見る  
3. `<Space>lr` でリネーム  
4. `<Space>lA` で import 整理  
5. `<Space>lf` で整形
